현재 goorm ide에 라이브러리가 적용되지 않습니다.
게시판은 한 페이지에 10개의 글을 노출시키도록 페이징 처리 되어있습니다.
글 번호, 제목, 댓글 갯수, 첨부파일 갯수, 작성자, 등록일을 확인할 수 있습니다.
검색 기능으로 제목, 내용, 작성자 키워드를 조합하여 검색 가능합니다.
게시판의 글 조회는 비회원으로도 가능하며, 글을 작성하려면 로그인을 해야합니다.
로그인 링크나 LOG IN 버튼을 눌러 로그인 가능합니다.
일반 로그인, 소셜 로그인이 가능하며 일반 로그인시 테스트 계정을 사용합니다.
소셜 로그인시 자동 회원가입 되고 모든 기능 사용 가능합니다.
REGISTER 버튼을 눌러 글 쓰기가 가능합니다.
제목, 내용, 첨부파일 업로드가 가능하며, 이미지 파일의 경우 미리보기로 썸네일이 출력됩니다.
자신이 작성한 글일 경우, 해당 글 조회시 MODIFY 버튼이 나타나며 이를 눌러 글 수정이 가능합니다.
수정 화면에서 Modify 버튼을 누르면 변경사항이 적용되고, Remove 버튼을 누르면 글이 삭제됩니다.
각 글의 Reply Count는 댓글의 갯수를 나타내며, 해당 텍스트 버튼 클릭시 댓글이 노출됩니다.
Add Reply로 댓글 작성이 가능합니다.
해당 댓글을 클릭하면 삭제 또는 수정이 가능합니다.
